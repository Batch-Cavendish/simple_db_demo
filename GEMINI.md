# Project: SimpleDB (C Educational Database)
- Language: C (Modern C11/latest paradigm)
- Environment: Ubuntu (Standard library preferred)
- Tone: Highly educational. Every function must have comments explaining *why* a database uses this specific mechanism.
- Error Handling: Use status codes; avoid `exit()` or `abort()`.
- Function: 
  - Create a new database with custom fields.
  - Be able to perform CRUD operations on the newly created database.
